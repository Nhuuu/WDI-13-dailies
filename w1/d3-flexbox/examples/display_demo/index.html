Display

Part 1: Inline vs. Block

Block elements will always start on a new line. Inline elements will always appear to continue on the same line as their neighbors.

** Notice the anchor element sits inline with the rest of the text.

Part 2: Dimension - A key difference between inline and block

An inline element will accept margin and padding, but the element still sits inline as you might expect. Margin and padding will only push other elements horizontally away, not vertically. It will ignore width and height.

Part 3: So what does the inline-block look like?

An element set to inline-block is very similar to inline in that it will set inline with the natural flow of text (on the "baseline"). The difference is that you are able to set a width and height which will be respected.

Part 4: Why is my text-align:center not working?
